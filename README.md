# Intro

Scaffold for experiments. Currently just:

- [NextJS](https://nextjs.org/docs)
- [Typescript](https://www.typescriptlang.org/docs/)
- [TailwindCSS](https://tailwindcss.com/docs)

# Development

To run

```
yarn install
yarn dev
```

To lint

```
yarn lint
```

To add a package

```
yarn add [-D] {package name(s)}
```
